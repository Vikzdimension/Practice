Indirect Recursion

Indirect Recursion is a type of recursion in which there are two or more functions and they call each other in a circular manner. In this type of recursion, the functions call each other in a circular manner. The function calls itself and then calls the other function and this process goes on.

Example

Let's take an example of two functions BuyBanana and BuyMore. Function BuyBanana calls function BuyMore and function BuyMore calls function BuyBanana. This is an example of indirect recursion.
